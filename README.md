# Instalação

Usando CURL
```
curl -sSL https://raw.githubusercontent.com/robertofgama/darth-instaulous/master/instalador.sh | sh
```

Usando WGET
```
wget -qO- https://raw.githubusercontent.com/robertofgama/darth-instaulous/master/instalador.sh | sh
```
